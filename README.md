# Clinic Queue Management System

A streamlined healthcare management platform designed to handle clinic workflows, user registrations (Doctors, Receptionists, Patients), and appointment queuing.

## 🚀 Live API Documentation
The backend for this project is hosted and documented here:
[View API Docs](https://cmsback.sampaarsh.cloud/api-docs/#/)

## ✨ Key Features
* **Role-Based Access:** Managed dashboards for Admins, Doctors, Receptionists, and Patients.
* **Clinic Enrollment:** Unique clinic codes for easy patient/staff onboarding.
* **User Management:** Internal creation of staff accounts with instant credentials.
* **Appointment Queuing:** Real-time tracking of patient visits and prescriptions.
* **Responsive UI:** Built with a clean, modern aesthetic inspired by NiceAdmin.

## 🛠️ Tech Stack
* **Frontend:** ASP.NET Core MVC, Bootstrap 5, Razor Pages.
* **Backend:** RESTful API (Node.js/C# - *adjust based on your actual backend*).
* **Data Handling:** JSON serialization with CamelCase support.
* **Styling:** Custom CSS for a professional medical-grade interface.
